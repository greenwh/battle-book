# Events Hub

Every timed event in Whiteout Survival, organized by frequency. Each guide follows the same structure: at-a-glance summary, mechanics, scoring, phase-by-phase actions, common mistakes, and a quick reference card.

## Currently Available

### State of Power (SvS)
**Monthly · Cross-state war**

The most rewarding event in the game. 5.5 days of preparation scoring against a matched enemy state, followed by a 12-hour battle phase, then 48 hours of troop revival.

- [SvS Overview & Strategy](/events/svs/overview)
- [SvS Prep Checklist](/events/svs/prep-checklist)

## Where to Start

**New to SvS?** Read the [SvS Overview](/events/svs/overview) front to back. It's the longest guide on the site, but SvS is the most rewarding event and the most punishing if you don't know the mechanics.

**Mid-event reference?** Use the [Prep Checklist](/events/svs/prep-checklist) — designed to be tapped through on mobile while you're actively playing.

**Looking for a specific mechanic?** Use the search box at the top — "polar terror," "field triage," "rebirth tomes" all jump to the right page.
